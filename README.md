BASKETBALL SHOT TRACKER FOR GARMIN WATCH
ONLY COMPATIBLE WITH A GARMIN FORERUNNER 55

There were no available apps on the Garmin ConnectIQ store to efficently track your shooting for basketball practice, so I taught myself Monkey C and devloped this application.

To run, copy the BasketballTracker.prg file into the APPS folder on your Garmin Watch.
Press START on your watch and find the Basketball Tracker app.
Upon running, you may track a make by pressing START, or record a miss pressing BACK.
Press DOWN to start a new lap.
Hold UP to open the menu.
Select Specific Session to start logging shots from a specific location on the court.
At any point you may go to menu => Display Stats to find a list of all locations and your accuracy for each location.
Menu => Exit to leave the app, session data will not save as of last update.
